# Python Loop Examples for DNA and Protein Sequences

This folder contains simple, clean Python scripts demonstrating how to use
loops (`while`, `for`, `range`) in the context of biological sequence analysis.

Scripts included:

- `01_find_donor_sites.py`: Find “gt” donor splice sites.
- `02_iterate_motifs.py`: Loop through DNA motifs.
- `03_range_examples.py`: Demonstrate range() (0–3, odd numbers).
- `04_protein_validator_all_positions.py`: Report all invalid amino acids.
- `05_protein_validator_break.py`: Stop at the first invalid amino acid.
- `06_protein_remove_invalid_continue.py`: Remove invalid amino acids.
- `07_prime_numbers_with_else.py`: Using `else` with for loops.
- `08_pass_statement_example.py`: Using `pass` as a placeholder.
