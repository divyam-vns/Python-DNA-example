# Stop Codon Checker (Python)


This folder contains Python scripts to detect **in‑frame stop codons** (TGA, TAG, TAA) in a DNA sequence.
It includes:
- A Boolean function `has_stop_codon()`
- Support for reading frames 0, 1, and 2
- Default argument handling
- A command‑line interactive script
- Test examples

````
Stop codons checked:
- TGA
- TAG
- TAA

The function supports specifying the **reading frame (0, 1, or 2)**.

## How It Works
The script reads a DNA sequence from user input or a file and determines whether it contains an in-frame stop codon.

## Usage
Run normally:
```
python3 has_stop_codon.py
python3 main.py
````
